# IMDB Movie Review Sentiment Analysis Using ML Models 

## Overview 

This project mainly works as text classification where IMDB reviews are classified as positive or negative review using ML techniques. 

The complete NLP pipeline is as follows:
- Data loading and verification 
- Text pre-processing :
 - html tag remov
 - lowercasing
 - stopword removal
 - tokenization 
 - lemmatizaiton
 - punctuation removal 
- Feature Extraction:
 - Unigram
 - Bigram
 - BoW
- Model building and training 
- Comparision with other ML algorithms 

## Dataset 
Name : IMDB Review 
- Total Samples : 50, 000
- Classes :
 -positive 
 -negative

Dataset Source : https://ai.stanford.edu/~amaas/data/sentiment/

## Problem Statement 

To predict whether the given review has positive or negative sentiment
